# Sutura Maps 🗺️

Plateforme de cartographie automatique des communes sénégalaises.
Génère une carte de localisation professionnelle en 30 secondes.

## Stack

- **Frontend** : HTML, CSS, JavaScript vanilla
- **Cartographie** : Leaflet.js + Turf.js
- **Export** : dom-to-image
- **Paiement** : Wave Mobile Money (lien statique → PayDunya API en cours)
- **Déploiement** : Netlify + Netlify Functions
- **Base de données** : Supabase (tokens de paiement en cours)
